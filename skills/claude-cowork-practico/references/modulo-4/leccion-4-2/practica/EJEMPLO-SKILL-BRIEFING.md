# Skill: Briefing de contenido para redes sociales

## Cuándo usar esta skill
Cuando necesites crear un briefing de contenido para un redactor o diseñador.
Esta skill genera briefings consistentes con el tono y estructura que usamos en el equipo.

## Contexto que necesita Claude
- El tema o producto del contenido
- La red social (Instagram, LinkedIn, TikTok)
- El objetivo (awareness, conversión, engagement)
- Cualquier restricción del cliente (colores, palabras prohibidas, tono)

## Instrucciones para Claude

Cuando generes un briefing de contenido, sigue siempre esta estructura:

**1. Objetivo de la pieza**
Una frase. Qué debe lograr esta pieza (ej: "Generar tráfico al producto X").

**2. Audiencia objetivo**
Quién lo va a ver. Datos demográficos y psicográficos relevantes.

**3. Mensaje principal**
La idea central en una frase. Todo lo demás es secundario a esta idea.

**4. Copy sugerido**
- Caption principal (máximo 150 palabras para LinkedIn, 80 para Instagram)
- Hashtags (máximo 5, relevantes, no genéricos)
- CTA claro al final

**5. Dirección visual**
- Tipo de imagen o video que funciona para este mensaje
- Referencias si hay (ej: "estilo similar a campaña X del mes anterior")
- Lo que NO hacer visualmente

**6. Métricas de éxito**
Cómo vamos a medir si esta pieza funcionó.

## Tono
Directo. Sin palabras vacías. Sin "increíble", "revolucionario" o "¡¡".
El lector es un profesional ocupado. Respeta su tiempo.

## Ejemplo de output esperado
[El briefing generado debe tener entre 200 y 350 palabras. No más.]
