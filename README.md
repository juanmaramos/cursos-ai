# Claude Cowork Práctico

Curso interactivo de Claude Cowork en español. Enseña a profesionales sin perfil técnico a delegar trabajo real a Cowork.

## Cómo usar

1. Instala el plugin en Cowork
2. Abre una nueva sesión
3. Escribe **"comencemos"** para iniciar el curso

## Contenido

- 6 módulos, 16 lecciones
- ~4 horas de práctica real con archivos incluidos
- Patrones aplicables a cualquier rol: PMs, founders, marketing, ops, finanzas, RRHH
