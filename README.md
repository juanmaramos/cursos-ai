# Cursos AI

Marketplace de plugins de cursos de IA para Claude.

## Plugins

- [Claude Cowork Practico](./claude-cowork-practico) - curso interactivo de Claude Cowork en espanol.

## Estructura

Cada curso vive en su propia carpeta de primer nivel. El archivo `.claude-plugin/marketplace.json` del repositorio registra los plugins disponibles para que Claude pueda instalarlos desde este repo como marketplace.
